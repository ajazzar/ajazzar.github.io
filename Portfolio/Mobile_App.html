I made this web page in my Mobile App class.I implemented a password-based entry page and a page to get basic information about the boiler such as the boiler ID, date of purchase, maximum allowable values of pressure and temperature, and an ability to change the password. I created a page to allow the user to change the basic information about the boiler and a page to enter data — temperature and pressure. In this Game, the user needs to click on the UFOs to destroy them. Using jQuery, the user clicks on the page button in order to swing the bat and hit the baseball in this game. Even though the course covered mobile design using jQuery UI, a framework called Bootstrap is used more widely in the industry to make responsive web pages. I used Bootstrap in order to display the nine Skywalker Saga Star Wars movies over three different pages in this Site. I used Flexbox in order to format this page based on my favorite childhood hobby, which was playing Nintendo and Super Nintendo games. When the user clicks on this page, the picture increases in size from 50 pixels to 500 pixels and the font size of the title increases, too. There are two images - a picture of "Mars Today" and a picture of "Ancient Mars" when liquid water flowed across the planet billions of years ago.
